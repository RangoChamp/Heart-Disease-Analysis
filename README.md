Heart Disease Data Analytics Dashboard

Overview
This project analyzes heart disease data to identify important health factors that contribute to cardiovascular disease. The analysis helps understand relationships between age, cholesterol levels, chest pain types, and heart disease risk.

This project was developed as part of the SkillWallet Data Analytics Program.

---

Objectives
- Analyze heart disease dataset
- Perform exploratory data analysis (EDA)
- Visualize important health indicators
- Build a dashboard to understand patterns in the data

---

Dataset
The dataset used in this project is the Heart Disease Dataset which contains several medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- Target (Heart Disease Presence)

Target values:
- **0 → No Heart Disease**
- **1 → Heart Disease**
